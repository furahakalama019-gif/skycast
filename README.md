SkyCast is a modern, responsive weather dashboard built to provide real-time weather updates and a 5-day forecast for any city worldwide. It features a sleek glassmorphism UI with dynamic dark mode support.

## ✨ Features
* **Real-time Data:** Fetches current temperature, weather conditions, and local time using the OpenWeather API.
* **5-Day Forecast:** Provides a clear breakdown of upcoming weather trends with custom-styled forecast cards.
* **Glassmorphism UI:** A clean, modern aesthetic with frosted glass effects and high-contrast typography.
* **Dark Mode:** Optimized for low-light environments with a deep midnight-blue theme.
* **Responsive Design:** Fully functional on desktop, tablet, and mobile devices.

## 🛠️ Tech Stack
* **HTML5** - Semantic structure
* **CSS3** - Custom styling, Flexbox, and Glassmorphism effects
* **JavaScript (ES6)** - Asynchronous API handling and DOM manipulation
* **OpenWeather API** - Weather data source
